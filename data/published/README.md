# Published Benchmark Data

This directory contains the published benchmark tables as machine-readable CSV files.

## Files

| File | Source Table | Description |
|------|-------------|-------------|
| `table6_measurements.csv` | Table 6 | Primary measurements (mean, SD) for all tasks × paradigms |
| `table8_lcri.csv` | Table 8 | Lifecycle Cost Redistribution Index by paradigm and task |
| `table10_raw_trials.csv` | Table 10 | Per-trial raw measurement data (all 60 trials) |
| `table5_coordination.csv` | Table 5 | Coordination properties by task |
| `table12_coordination_matrix.csv` | Table 12 | Coordination complexity matrix (exact counts) |
| `table7_dominant_cost.csv` | Table 7 | Dominant cost component by paradigm and task |

All values are transcribed directly from the published paper. No fabricated or synthetic data.
