# Roadmap

## Short-Term (0–6 months)

- [ ] Independent replication by third-party researchers
- [ ] Multi-operator replication protocol development
- [ ] Additional LLM evaluation (Claude, Gemini, open-source models)
- [ ] Additional visual platform evaluation (Make, Node-RED, Zapier)

## Medium-Term (6–18 months)

- [ ] Expanded benchmark suite (12–16 tasks)
- [ ] Longitudinal field studies in enterprise environments
- [ ] Automated telemetry collection pipeline
- [ ] Paradigm hybridization experiments
- [ ] Formal modeling of lifecycle-effort redistribution

## Long-Term (18+ months)

- [ ] Community-maintained benchmark repository
- [ ] CI/CD integration for automated replication
- [ ] Standardized LCRI reporting format
- [ ] Integration with third-party workflow evaluation frameworks

See the paper's Future Research Agenda (Section 8.2) for detailed descriptions.
