# Published Tables

This directory contains machine-readable CSV exports of all tables from the paper.

## Table Files

| File | Description |
|------|-------------|
| `table2_research_gap_matrix.csv` | Research gap matrix positioning this work relative to prior studies |
| `table3_task_suite.csv` | Benchmark task suite summary |
| `table4_metric_scopes.csv` | Activities included/excluded per metric |
| `table5_coordination_properties.csv` | Coordination properties by task |
| `table6_primary_measurements.csv` | Recorded metrics (IC, MC, RC, VC, TWC) with SD |
| `table7_dominant_cost.csv` | Dominant cost component by paradigm and task |
| `table8_lcri.csv` | Lifecycle Cost Redistribution Index by paradigm and task |
| `table9_alternative_explanations.csv` | Alternative explanations and their potential impact |
| `table10_raw_measurements.csv` | Per-trial raw measurements (all 60 trials) |
| `table11_research_gap_matrix_detailed.csv` | Research gap matrix — detailed |
| `table12_coordination_complexity_matrix.csv` | Coordination complexity matrix |
| `table13_evidence_synthesis.csv` | Summary of evidence for each finding |

All values are transcribed directly from the published paper. No fabricated or synthetic data.
