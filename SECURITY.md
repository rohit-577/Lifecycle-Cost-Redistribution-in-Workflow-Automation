# Security Policy

## Reporting a Vulnerability

This repository contains research artifacts and benchmark data. If you discover a security vulnerability in the code or scripts provided:

1. **Do not** open a public issue.
2. Contact the author directly at rohit.r@pes.edu.
3. Provide a detailed description of the vulnerability and steps to reproduce.

## Scope

This is an academic research repository. The code is provided as-is for reproducibility purposes and is not intended for production deployment.

## Safe Use

- The benchmark scripts are designed for controlled laboratory environments.
- API keys and credentials used in workflows are placeholder values.
- Do not deploy example workflows against production systems without review.
