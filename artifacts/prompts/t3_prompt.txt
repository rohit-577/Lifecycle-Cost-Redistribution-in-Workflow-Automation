You are a workflow automation expert. Create a Python script that processes a JSON payload containing fields type (one of: order, refund, inquiry, escalation), priority (high/medium/low), and amount (numeric). (4) If type is 'inquiry', route to 'support_triage'. (5) If type is 'escalation', route to 'escalation_queue' regardless of amount. Priority overrides: if priority is 'high' for any type, add a 'priority_overlay' flag. If amount is missing or non-numeric, set it to 0 and add a 'default_applied' warning. Return the transformed payload with a route field indicating the path taken.
